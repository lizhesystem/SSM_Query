### SSM用户综合查询

实现功能：
用户注册、登录、分页查询从数据库获取的信息。
 
涉及细节：
展示页面有增删改查四个功能,分页过滤搜索。

运行环境
- 运行环境：Wind10
- jdk：1.9
- tomcat：8.0.x
- spring版本：5.2.0
- mybatis版本：3.4.5
- 数据库版本：mysql 8.1

基于Maven搭建,很简单的一个ssm入门增删改查的一个项目,前端页面基于`bootstrap`写的几个简单的页面,适合初学者了解整个SSM整合的配置和代码结构。

#### 访问入地址入口
> http://localhost/pages/index.jsp
